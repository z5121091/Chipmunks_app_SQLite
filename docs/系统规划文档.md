### 2.6 扫码查库存（库位一维码）

```
┌─────────────────────────────────────────────────────────────┐
│                    扫码查库存流程                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ① 选择仓库（可选）                                         │
│       ↓                                                      │
│  ② 扫描库位一维码（贴在货架上的型号标签）                   │
│       ↓                                                      │
│  ┌─────────────────────┐                                    │
│  │ 扫码内容：ABC-123   │                                    │
│  └─────────────────────┘ │                                    │
│       ↓                                                      │
│  ③ 查询 ERP 库存                                              │
│       ↓                                                      │
│  ┌─────────────────────┐                                    │
│  │ 型号：ABC-123        │                                    │
│  │ ─────────────────── │                                    │
│  │ 总库存：180 个       │                                    │
│  │                      │                                    │
│  │ 仓库A：100个        │                                    │
│  │ 仓库B： 80个        │                                    │
│  └─────────────────────┘                                    │
│                                                              │
│  一维码格式：根据 ERP 物料编码确定（后续确认）                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

**特点**：
- 扫码即查，无需手动输入
- 一维码贴在库位上，拿起 PDA 一扫就知道库存
- **直连 ERP API**，实时查询最新库存
- 可选仓库范围

**实现方式**：
```
扫码 → 调用 ERP 库存查询 API → 返回结果显示
```

**核心价值**：扫码效率 + 现场操作 + 防错

## 六、功能优先级

### 6.1 APP 端优先级

| 优先级 | 功能 | 原因 |
|--------|------|------|
| P0 | ERP 对接基础框架 | 没有它其他都是空谈 |
| P1 | 扫码入库（匹配采购单） | 核心业务 |
| P1 | 扫码出库（匹配销售单） | 核心业务 |
| P1 | 扫码防错验证 | 减少错误 |
| P2 | 实时库存查询 | 方便快捷 |
| P2 | 扫码查库存（库位一维码） | 快速定位库存 |
| P2 | 拆包自动标签 | 提高效率 |
| P2 | 盘点功能 | 需要时用 |
| P3 | 库存预警 | 预防断货 |
| P3 | FIFO 提示 | 减少损耗 |

### 6.2 桌面端优先级

| 优先级 | 功能 | 原因 |
|--------|------|------|
| P0 | 同步服务 | 数据中转核心 |
| P1 | 数据管理（订单/物料查询） | 日常工作 |
| P1 | 客户信息管理 | 发货必备 |
| P1 | 待发货订单列表 | 发货流程 |
| P2 | 顺丰文件生成 | 减少人工输入 |
| P2 | 标签打印模板 | 刚需 |
| P3 | 短信通知 | 客户体验 |
| P3 | 数据统计报表 | 管理需求 |
| P4 | 多操作员/权限 | 多人协作 |
