# 掌上仓库 - 使用说明

## 一、应用简介

掌上仓库是一款仓库管理移动应用，支持扫码入库、出库发货、库存盘点、物料管理等功能，可通过电脑同步服务将数据导出到Excel文件。

**主要功能：**
- 扫码入库：扫描二维码快速入库
- 扫码出库：关联订单发货出库
- 库存盘点：整包/拆包两种盘点模式
- 物料管理：型号与存货编码绑定
- 数据同步：一键导出到电脑

---

## 二、首页

### 2.1 功能入口

首页展示五大功能模块入口：

| 模块 | 说明 |
|------|------|
| 扫码入库 | 扫描物料二维码入库，按型号自动累加数量 |
| 扫码出库 | 扫描物料出库，关联订单进行发货 |
| 盘点 | 库存盘点，支持整包盘点和拆包盘点两种模式 |
| 物料管理 | 管理型号与存货编码的绑定关系 |
| 设置 | 仓库管理、解析规则、数据同步、备份恢复等 |

### 2.2 统计数据

首页顶部显示数据统计：
- 总订单数 / 今日新增订单
- 总物料数 / 今日新增物料
- 总数量 / 今日入库数量

## 三、扫码入库

### 3.1 功能说明

扫描物料二维码进行入库操作，相同型号的物料数量会自动累加。

### 3.2 操作步骤

1. 选择目标仓库（默认自动选择默认仓库）
2. 点击扫描区域或手动输入数据
3. 扫描物料二维码，系统自动解析型号、批次、数量
4. 确认入库，入库记录保存到数据库

### 3.3 仓库切换

- 点击仓库选择器可切换目标仓库
- 仓库在【设置 → 仓库管理】中添加

### 3.4 手动输入

- 如无法扫描，可点击"手动输入"按钮
- 输入二维码内容后点击确认

---

## 四、扫码出库

### 4.1 功能说明

扫描物料进行出库操作，需关联订单进行发货。

### 4.2 操作流程

**步骤一：选择/创建订单**
- 首次扫描会提示选择订单
- 可选择已有订单或创建新订单
- 订单信息包含：订单号、客户名称

**步骤二：扫描物料**
- 扫描物料二维码，系统自动解析信息
- 物料自动关联到当前订单
- 显示已扫描物料列表

**步骤三：完成出库**
- 确认物料信息无误
- 点击保存完成出库操作

### 4.3 订单管理入口

- 点击"订单"按钮可进入订单管理页面
- 可查看、搜索、删除订单
- 可展开订单查看关联物料

### 4.4 标签管理入口

- 点击"标签"按钮可进入标签管理页面
- 查看拆包生成的标签记录
- 支持批量删除标签

---

## 五、盘点

### 5.1 功能说明

进行库存盘点，支持两种模式：
- **整包盘点**：盘点完整的物料包
- **拆包盘点**：将物料包拆分成小包

### 5.2 整包盘点

1. 选择盘点模式为"整包"
2. 选择目标仓库
3. 扫描物料二维码
4. 系统记录盘点数量
5. 确认保存盘点记录

### 5.3 拆包盘点

1. 选择盘点模式为"拆包"
2. 选择目标仓库
3. 扫描物料二维码
4. 弹出数量输入框，输入实际数量
5. 系统生成拆包标签
6. 继续扫描或完成盘点

### 5.4 拆包标签

拆包后会生成标签记录，包含：
- 标签类型（发货标签/剩余标签）
- 订单号、客户、型号、批次
- 原数量、新数量
- 追踪码、箱号
- 拆包时间

---

## 六、物料管理

### 6.1 功能说明

管理物料型号与存货编码的绑定关系，用于库存系统对接。

### 6.2 新增绑定

1. 点击右上角"+"按钮
2. 填写绑定信息：
   - 扫描型号：扫描二维码时识别的型号
   - 存货编码：对应库存系统的编码
   - 供应商：供应商名称（可选）
   - 备注：其他说明（可选）
3. 点击保存

### 6.3 编辑绑定

- 点击绑定记录进入编辑模式
- 修改后点击保存

### 6.4 删除绑定

- 左滑绑定记录
- 点击删除按钮

### 6.5 导入导出

**导出模板：**
- 点击"导出模板"获取Excel模板文件

**导入数据：**
- 点击"导入"按钮
- 选择填写好的Excel文件
- 系统自动导入绑定数据

---

## 七、设置

设置页面包含六大功能分组：

### 7.1 基础配置

#### 7.1.1 仓库管理

管理仓库信息，支持多仓库。

**新增仓库：**
1. 点击"新增仓库"
2. 填写仓库名称、描述
3. 可设置是否为默认仓库
4. 点击保存

**编辑仓库：**
- 点击仓库记录进入编辑模式

**删除仓库：**
- 左滑仓库记录
- 点击删除按钮

**设置默认仓库：**
- 编辑仓库时开启"默认仓库"开关
- 默认仓库会在入库/盘点时自动选中

#### 7.1.2 自定义字段

配置解析规则时可使用的自定义字段。

**新增字段：**
1. 点击"新增字段"
2. 填写字段名称和键名
3. 点击保存

**编辑字段：**
- 点击字段记录进入编辑模式

**删除字段：**
- 左滑字段记录点击删除

---

### 7.2 解析配置

#### 7.2.1 解析规则

配置二维码解析规则，系统根据规则自动解析扫描内容。

**新增规则：**
1. 点击"新增规则"
2. 填写规则信息：
   - 规则名称：便于识别的名称
   - 规则描述：规则说明
   - 分隔符：数据分隔方式（默认 "/" 或自定义）
   - 字段映射：设置每个位置对应的字段
   - 匹配条件：根据特定位置的关键字自动识别规则
3. 点击保存

**字段映射说明：**
- 选择分隔符后，系统按位置解析
- 每个位置可映射到对应字段
- 支持系统字段和自定义字段

**匹配条件：**
- 设置特定位置包含的关键字
- 扫描时自动匹配对应规则
- 无匹配时使用默认规则

**编辑规则：**
- 点击规则卡片进入编辑模式

**启用/禁用规则：**
- 点击规则开关切换状态
- 禁用的规则不会被自动匹配

**删除规则：**
- 点击规则卡片右上角删除按钮

---

### 7.3 数据同步

将数据同步到电脑，需要先运行电脑同步服务。

#### 7.3.1 配置服务器

1. 输入电脑IP地址（如：192.168.1.100）
2. 端口默认8080
3. 点击"测试连接"确认连通性
4. 点击保存配置

#### 7.3.2 同步数据

**入库单同步：**
- 点击"同步入库单"
- 将入库记录导出到电脑
- 保存为 `入库单.xlsx`

**出库单同步：**
- 点击"同步出库单"
- 将出库记录导出到电脑
- 保存为 `出库单.xlsx`

**盘点单同步：**
- 点击"同步盘点单"
- 将盘点记录导出到电脑
- 保存为 `盘点单.xlsx`

**标签数据同步：**
- 点击"同步标签数据"
- 将拆包标签导出到电脑
- 保存为 `标签打印.xlsx`

#### 7.3.3 连接状态

- 连接成功：显示绿色状态
- 连接失败：显示红色状态，请检查网络和服务器
- 心跳检测：自动检测连接状态

---

### 7.4 备份恢复

#### 7.4.1 数据备份

1. 点击"备份数据"
2. 选择保存位置
3. 生成JSON备份文件
4. 备份内容包括：
   - 解析规则
   - 自定义字段
   - 物料绑定
   - 仓库配置

#### 7.4.2 数据恢复

1. 点击"恢复数据"
2. 选择备份JSON文件
3. 确认恢复
4. 恢复后会覆盖当前配置

---

### 7.5 数据管理

#### 7.5.1 清空数据

**清空入库记录：**
- 点击"清空入库记录"
- 确认后删除所有入库数据

**清空出库记录：**
- 点击"清空出库记录"
- 确认后删除所有订单和物料数据

**清空盘点记录：**
- 点击"清空盘点记录"
- 确认后删除所有盘点数据

**清空标签记录：**
- 点击"清空标签记录"
- 确认后删除所有拆包标签

**清空全部数据：**
- 点击"清空全部数据"
- 确认后删除所有业务数据
- 配置数据（规则、字段、仓库）会保留

## 八、电脑同步服务

### 8.1 安装运行

**安装依赖：**
```bash
pip install cherrypy openpyxl pystray Pillow pyinstaller
```

**运行服务：**
```bash
python label_sync_server.py
```

**打包成exe：**
```bash
python -m PyInstaller --onefile --windowed --name "掌上仓库同步服务" --icon "icon.ico" label_sync_server.py
```

或直接运行 `build.bat`

### 8.2 服务功能

- 系统托盘运行，不占任务栏
- 支持开机自启动
- 自动接收APP发送的数据
- 保存为带格式的Excel文件
- 提供健康检查端点 `/health` 供APP检测服务状态

### 8.3 数据保存位置

所有数据统一保存到 `D:/数据同步/` 目录：

| 文件名 | 说明 |
|--------|------|
| 发货序列号.xlsx | 扫码序列号记录 |
| 入库单.xlsx | 入库记录 |
| 出库单.xlsx | 出库记录 |
| 盘点单.xlsx | 盘点记录 |
| 标签打印.xlsx | 标签数据 |
| 物料数据.xlsx | 物料绑定数据 |

### 8.4 托盘菜单

右键托盘图标可：
- 设置开机自启动
- 查看服务器IP和端口
- 打开数据目录
- 查看日志
- 退出服务

## 九、常见问题

### 9.1 扫描无反应

- 检查相机权限是否开启
- 确认二维码格式正确
- 尝试手动输入

### 9.2 解析失败

- 检查解析规则是否正确配置
- 确认分隔符设置与实际匹配
- 查看是否有匹配条件冲突

### 9.3 无法连接同步服务

**基础检查：**
- 确认电脑和手机在同一WiFi网络下
- 检查防火墙是否放行8080端口
- 确认同步服务正在运行（系统托盘有图标）
- 检查IP地址是否正确（查看托盘菜单显示的IP）

**详细排查步骤：**
1. 在电脑浏览器访问 `http://localhost:8080/health`，应返回 `{"status":"ok",...}`
2. 如果本机能访问但手机不能，检查防火墙设置
3. 关闭Windows防火墙测试，确认是否为防火墙问题
4. 如果防火墙问题，添加入站规则放行8080端口

**防火墙设置（Windows）：**
1. 打开"高级安全Windows Defender防火墙"
2. 点击"入站规则" → "新建规则"
3. 选择"端口" → "TCP" → "特定本地端口: 8080"
4. 选择"允许连接"
5. 应用到所有配置文件
6. 命名为"掌上仓库同步服务"

### 9.4 数据丢失

- 定期使用备份功能
- 备份文件妥善保存
- 可通过恢复功能还原
