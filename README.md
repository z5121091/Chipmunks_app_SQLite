## ✨ 核心功能

### 1️⃣ 扫码入库
- **仓库选择**：进入页面后选择目标仓库，切换仓库自动保存当前数据
- **扫码录入**：支持PDA扫描枪扫描物料二维码，自动识别并提取型号信息
- **供应商自动带出**：型号已在物料管理中维护时自动带出供应商
- **重复扫码处理**：检测到重复扫码时提示，可选择作废重复数据
- **同型号聚合**：同一型号多次扫描时数量自动合并相加
- **数据管理**：仅缓存数据，支持导出入库单，可查看历史入库记录
- **确认入库**：点击确认入库保存记录，自动生成入库单号

### 2️⃣ 扫码出库
- **仓库选择**：进入页面后选择目标仓库，切换仓库自动保存当前数据
- **扫码识别**：可扫描物料二维码或订单码，系统自动识别内容类型
- **订单自动处理**：扫描订单码时自动新建或切换订单，扫描物料时自动关联到当前订单
- **数据累加**：同一物料多次扫描时数量自动累加
- **操作限制**：出库数据仅支持删除，不可编辑、不可拆包
- **重复提示**：检测到重复扫码时震动提示
- **导出功能**：支持导出出库单数据

### 3️⃣ 订单管理
- **订单列表**：展示所有出库订单，支持按状态筛选（全部/待发货/已发货）
- **创建订单**：支持手动创建订单，输入订单号和客户名称
- **拆包功能**：订单管理是唯一的拆包入口，适用于整包货品客户只拿部分数量的情况
- **执行拆包**：输入发货数量后自动生成发货标签和剩余标签
- **标签管理**：发货标签数量等于输入的发货数量，剩余标签数量为原数量减去发货数量
- **标签操作**：单条标签可单独导出或暂存后续处理
- **查看详情**：查看订单物料信息，导出订单数据，删除订单

### 4️⃣ 盘点管理
- **仓库选择**：进入页面后选择目标仓库，切换仓库自动保存当前数据
- **盘点模式**：支持整包盘点和拆包盘点两种模式
- **整包盘点**：扫描物料二维码自动识别并聚合，仅支持删除，不可修改数量
- **拆包盘点**：扫描物料二维码后录入实际库存数量，生成替换新标签
- **重复提示**：检测到重复扫码时震动提示
- **导出功能**：支持导出盘点结果，包含盘点修正后的拆包替换标签
- **保存盘点**：扫描完成后点击保存，盘点记录保存到数据库

### 5️⃣ 物料管理
- **功能说明**：维护型号与供应商的绑定关系，扫码时通过型号匹配自动带出供应商
- **录入方式**：支持手动逐条新增和模板导入/导出
- **维护字段**：型号、供应商、存货编码、备注
- **新增绑定**：点击右上角按钮新增绑定，填写信息后保存
- **编辑删除**：点击记录进入编辑，左滑记录可删除
- **导入导出**：导出Excel模板，填写后批量导入绑定数据

### 6️⃣ 系统设置
- **仓库管理**：新增、编辑、删除仓库，设置默认仓库
- **扫码提示音**：开启/关闭扫码成功、重复、错误时的语音播报和震动反馈
- **版本检测**：从NAS远程检测并下载最新版本APK
- **解析规则**：配置二维码解析规则，包含规则名称、分隔符、字段顺序
- **自定义字段**：扩展自定义字段，在解析规则中添加使用
- **数据同步**：手动填写服务器IP和端口，对接电脑同步脚本
- **备份恢复**：备份解析规则、自定义字段、物料绑定、仓库、同步服务器配置
- **数据管理**：清空订单数据、物料数据、标签缓存数据

## 🛠️ 技术架构

### 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| **前端** | Expo 54 + React Native | 移动端应用开发 |
| **后端** | Express.js | 服务端 API |
| **数据库** | SQLite | 本地数据存储 |
| **存储** | Expo FileSystem | 文件系统管理 |
| **UI** | Tailwind CSS | 样式系统 |

### 架构特点

- **Monorepo 结构**：前后端代码统一管理
- **纯离线运行**：无需网络连接，数据本地存储
- **高性能 SQLite**：移动端优化，快速响应
- **PDA 设备支持**：专业扫码设备兼容
- **现代化 UI**：Tailwind CSS + Uniwind

## 🚀 快速开始

### 安装应用

1. 下载 APK 安装包
2. 在 Android 设备上安装应用
3. 首次使用需要授予存储权限、相机权限等

### 首次使用

1. **添加仓库**：进入系统设置 → 仓库管理 → 添加仓库
2. **配置解析规则**（可选）：进入系统设置 → 解析规则 → 配置二维码解析规则
3. **维护物料绑定**（可选）：进入物料管理 → 新增型号与供应商绑定
4. **开始使用**：从首页选择功能入口开始使用

### 主要功能入口

- **扫码入库**：扫描物料二维码，录入入库信息
- **扫码出库**：扫描物料二维码或订单码，执行出库操作
- **订单管理**：管理出库订单，执行拆包操作
- **盘点管理**：进行仓库盘点，记录盘点数据
- **物料管理**：维护型号与供应商绑定关系
- **系统设置**：配置仓库、解析规则、数据同步等
